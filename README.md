Edt Generation
===
    Test pour la génération des feuilles de présence

### Requirements
- python3
- pyp3

### Install requirements
If you live in a cave :
```
$ sudo apt-get update
$ sudo apt-get install python3 python3-pip
```

Install python libraries:
```
$ pip3 install -r requirements.txt
```

### Run it
```
$ python3 run.py
```
